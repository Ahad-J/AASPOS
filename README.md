## 🚀 Getting Started

### Prerequisites
- Node.js v16+
- Python 3.8+
- PostgreSQL 12+

### Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/yourproject.git
```

2. Install dependencies
```bash
cd yourproject
npm install
```

3. Setup environment variables
```bash
cp .env.example .env
```

4. Start the development server
```bash
npm run dev
```
